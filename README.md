# Superstack

A Claude Code skill for building production-ready, full-stack web products with award-winning design, expert copywriting, IT security, testing, CI/CD, accessibility, GDPR compliance, and auto-deployment.

## What it does

Superstack turns Claude into a senior full-stack developer, creative director, security engineer, and DevOps specialist — all in one. It enforces a structured 15-phase workflow that takes a project from idea to deployed product without cutting corners.

### Philosophy

Every project built with Superstack should feel like it was crafted by a boutique agency, not generated by AI. No generic hero sections, no "Welcome to our platform" headlines, no cookie-cutter card grids. Instead: Lenis smooth scroll, editorial layouts, intentional whitespace, bold typography, OKLCH color system, noise/grain textures, custom icons, micro-interactions, and scroll-driven storytelling.

## The 15 Phases

| Phase | Name | What happens |
|:---:|---|---|
| **0** | **Deep Discovery & Concept Document** | Mandatory. Gathers ALL requirements across 10 groups: Product & Audience, Technical Foundation, Tracking & Marketing, E-Mail System, Payment & Monetization, Hosting & Deployment, Git & Development Workflow, Platform Features, Compliance & Legal, Design Direction. Outputs a `PROJECT-CONCEPT.md` with MoSCoW feature matrix. |
| **1** | **Copywriting Research** | Researches competitors and audience, maps 3-5 core pain points, crafts conversion-focused copy using PAS framework, headline formulas, and contextual CTAs. |
| **2** | **Design System** | Lenis smooth scroll, OKLCH color system, noise/grain textures, Framer Motion + GSAP animations, custom/curated icons (Phosphor, Tabler, Iconoir), Tailwind v4, modern CSS (container queries, `:has()`, scroll-driven animations). |
| **3** | **Full Scaffold** | Complete production-ready project with real copy, Lenis, responsive at all breakpoints, every UI state handled, animations wired up, noise textures, SEO, dark mode, back-end (API, DB, auth), Biome linting, README, CHANGELOG. |
| **4** | **IT Security Audit** | XSS, CSRF, IDOR, SQL injection, auth hardening, API security, webhook verification, dependency audit. Zero Critical/High vulnerabilities. |
| **5** | **Accessibility (WCAG 2.1 AA)** | Legally required in EU since June 2025 (EAA). Semantic HTML, keyboard navigation, ARIA, color contrast, reduced motion, 44px touch targets. |
| **6** | **Performance** | LCP < 2.5s, CLS < 0.1, INP < 200ms. Image optimization, font subsetting, code splitting, edge functions. |
| **7** | **Continuous Testing** | Vitest + Testing Library, Playwright E2E, MSW for API mocking. Tests written alongside features, not after. |
| **8** | **CI/CD Pipeline** | GitHub Actions: lint, type-check, test, build. Preview deploys, environment-specific deployment, Docker builds. |
| **9** | **Git Workflow & Documentation** | Conventional Commits, automatic changelog (semantic-release), README maintenance, API docs (OpenAPI/Swagger), branch protection. |
| **10** | **Tracking & Marketing** | Cookie consent (Klaro/Cookiebot), Google Consent Mode v2, GTM, Meta Pixel + CAPI with event deduplication, TikTok/LinkedIn pixels, privacy-friendly analytics. |
| **11** | **Analytics & Monitoring** | Sentry with tunnel route, Pino structured logging, health check endpoints, uptime monitoring. |
| **12** | **SEO + GEO** | Technical SEO, schema markup, GEO for AI search engines (ChatGPT, Perplexity, Google AI Overviews). |
| **13** | **Deployment** | Auto-deploy to Vercel, Dokploy, Docker. Multi-environment (DEV/STAGING/PROD). Environment variables per stage. |
| **14** | **Optional Extras** | Cost estimation, DSGVO-quality legal texts, extended documentation (API docs, Storybook, ADRs). |

## Key Features

- **10-group deep discovery** — Phase 0 asks about tracking pixels, email system, payment details, admin panel, environments, compliance, and more — so nothing gets forgotten.
- **PROJECT-CONCEPT.md** — Complete concept document with personas, MoSCoW feature matrix, architecture diagram, user flows, and decision log before any code is written.
- **Lenis smooth scroll** — Standard on every project. Synced with GSAP ScrollTrigger for premium scroll-driven animations.
- **OKLCH color system** — Perceptually uniform palettes. Derive entire palette from a single brand hue.
- **Noise/grain textures** — SVG noise overlays on hero sections and cards for depth. Breaks the "too clean" AI look.
- **Anti-AI design** — No symmetric grids, generic gradients, or default icons. Bento grids, asymmetric splits, editorial compositions, staggered animations.
- **Better Auth (2026 standard)** — Auth.js team joined Better Auth in Sept 2025. Recommended for all new projects.
- **Stripe Embedded Checkout** — Server Actions pattern, keeps users on your domain. Billing Portal, webhooks, EU VAT.
- **React Email + Resend** — Component-based email templates with SPF/DKIM/DMARC setup.
- **Google Consent Mode v2** — Mandatory. Consent defaults loaded BEFORE any tracking scripts.
- **Facebook CAPI** — Server-side tracking with event deduplication (same event_id client + server).
- **DDG-compliant Impressum** — German Digital-Dienste-Gesetz (replaced TMG since May 2024).
- **EAA accessibility** — European Accessibility Act in force since June 2025. Fines up to 100K EUR.
- **Biome** — Replaces ESLint + Prettier. Single tool for linting and formatting.
- **Multi-environment** — DEV/STAGING/PROD configs, environment-specific deployment, feature flags.

## Installation

### As a Plugin (recommended — includes slash command)

```bash
claude plugin add /path/to/superstack
```

Or from GitHub:

```bash
claude plugin add https://github.com/arrrda8/superstack
```

### As a Skill only

Copy `skills/superstack/` (with `SKILL.md` + `references/`) to `~/.claude/skills/superstack/` (global) or `your-project/.claude/skills/superstack/` (project-local).

## Usage

### Slash Command

```
/superstack:build
```

Or with a project description:

```
/superstack:build SaaS dashboard for project management with auth, Stripe billing, and GDPR compliance
```

### Automatic Triggering

The skill triggers automatically when you describe a web project:

- *"Build me a landing page for my SaaS product"*
- *"Erstell mir eine Webseite fuer mein Startup"*
- *"I need a customer dashboard with auth, billing, and GDPR compliance"*
- *"Make a full-stack MVP for my startup idea"*

## Tech Stack Flexibility

Superstack adapts to your choices during Phase 0:

- **Frameworks**: Next.js, Nuxt, SvelteKit, Astro
- **CSS**: Tailwind v4, CSS Modules
- **Databases**: Supabase (PostgreSQL), Firebase, PlanetScale, Convex
- **Auth**: Better Auth, Clerk, Supabase Auth
- **Hosting**: Vercel, Dokploy, Railway, Hetzner, AWS
- **Payments**: Stripe, LemonSqueezy, Paddle
- **Email**: Resend, Postmark, SendGrid
- **Analytics**: Plausible, Umami, PostHog, GA4
- **State**: Zustand + TanStack Query
- **Linting**: Biome (recommended), ESLint
