# Superstack Plugin

A Claude Code plugin for building production-ready, full-stack web products with award-winning design, expert copywriting, IT security, testing, CI/CD, accessibility, and auto-deployment.

## Installation

```bash
claude plugin install --path /path/to/superstack-plugin
```

Or from a marketplace:
```bash
claude plugin install superstack
```

## Usage

### Slash Command

```
/superstack:build
```

Or with a project description:

```
/superstack:build SaaS dashboard for project management with auth, Stripe billing, and GDPR compliance
```

### Automatic Triggering

The skill also triggers automatically when you describe a web project, e.g.:

- *"Build me a landing page for my SaaS product"*
- *"Create a portfolio website"*
- *"I need a customer dashboard with auth and billing"*

## What it does

12-phase workflow: Requirements → Copywriting → Design → Scaffold → Security → Accessibility → Performance → Testing → CI/CD → Analytics → SEO+GEO → Deployment + Optional Extras.

See the full README in the [skill repository](https://github.com/YOUR_USERNAME/superstack-skill) for details.
